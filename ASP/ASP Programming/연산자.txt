[대입 연산자]

기호 / 기능

= / 오른쪽에 값을 왼쪽에 변수에 대입

+= / 왼쪽 변수 값과 오른쪽의 값을 더한 후 결과를 왼쪽의 변수에 할당

-= / 왼쪽 변수 값과 오른쪽의 값을 뺀 후 결과를 왼쪽의 변수에 할당

*= / 왼쪽 변수 값과 오른쪽의 값을 곱한 후 결과를 왼쪽의 변수에 할당

/= / 왼쪽 변수 값과 오른쪽의 값을 나눈 후 결과를 왼쪽의 변수에 할당


[산술 연산자]


기호 / 의미

^ / 지수승

- / 단항 부정

* / 곱셈

/ / 나눗셈

\ / 나눗셈 (정수 결과를 반환)

Mod / 나머지 연산

+ / 덧셈

-/ 뺄셈

& / 스트링 연결 접속
