C 언어 등의 프로그래밍 언어에서는 변수의 타입에 대한 명시적인 선언을 하고
변수를 사용한다. 그러나 VBScript 등의 스크립트 언어는 엄격한 변수의 타입 선언을 요구하지 않는데, 
이러한  타입을 변형체라고 한다.
VBScript는 변형체라고 하는 하나의 데이터 타입만을 가진다. 변형체란 서로 다른 타입의 데이터를 수용할 수 있는
타입을 말한다. 특정 변수에 정수를 할당하면이 변수는 정수 값을 가지는 변수가 되며, 문자열을 할당하면 
문자열을 저장하는 변수가 된다. 변수의 타입을 선언할 필요 없이 원하는 값을 할당하기만 하면
할당된 값의 유형에 따라 변수의 타입이 결정되는 형태가 바로 변형체이다.
