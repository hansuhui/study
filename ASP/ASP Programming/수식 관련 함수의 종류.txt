함수 / 기능

Atn(수) / 아크 탄젠트 값을 반환

Cos(수) / 코사인 값을 반환

Sin(수) / 사인값을 반환

Tan(수) / 탄젠트 값을 반환

Exp(수) / 승수로 거듭제곱하는 자연로그의 밑이 el인 값을 반환

Log(수) / 자연로그를 반환

Sqr(수) / 제곱근을 반환

Hex(수) / 수를 Hex string으로 변환

Abs(수)  / 수의 절대값을 반환

Int(수) /  수의 정수부를 반환

Fix(수) / 수의 정수부를 변환

Oct(수) / 수를 Octal string으로 변환

Val(식) / 식을 수로 변환

Round(식) / 식을 반올림하여 정수를 반환

Randomize() / 난수 발생기를 초기화

Rnd() / 1 미만의 난수를 발생
